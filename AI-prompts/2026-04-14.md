## Code conventions (always follow)

| Concern | Rule |
|---|---|
| TypeScript | Strict, functional components, no `any` |
| Mobile state | Zustand (auth) + TanStack Query v5 |
| Dashboard state | react-query v3 — `useQuery('key', fn)` |
| Navigation (mobile) | Expo Router v3 |
| Screen exports | `export function Foo()` + `export default Foo` |
| Tab re-exports | `export { default } from '../../src/screens/XScreen'` |
